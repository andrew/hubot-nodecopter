# Hubot Nodecopter

Control a Nodecopter with Hubot

## Config

    HUBOT_IMGUR_API_KEY

## Commands

    hubot takeoff
    hubot land
    hubot forward
    hubot back
    hubot left
    hubot right
    hubot up
    hubot down
    hubot photo
    hubot clockwise
    hubot anticlockwise
    hubot stop
    hubot photo
    hubot status
